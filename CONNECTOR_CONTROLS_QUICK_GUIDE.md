### Change Connection Side
1. **Click** on a connector to select it
2. **Gray squares** appear at all four sides of both boxes
3. **Blue squares** show current connection points
4. **Drag a gray square** toward the side you want
5. Connector snaps to nearest side automatically

### Reset to Auto
1. **Right-click** on a connector
2. Select **"Reset to Auto Route"**
3. All custom bends removed
4. Returns to simple default path

## Troubleshooting

**Q: I don't see the blue circle**
A: Click the connector to select it first

**Q: The connector won't bend**
A: Make sure you're dragging the **blue circle**, not the line itself

**Q: How do I add multiple bends?**
A: The system uses single-bend design for cleaner diagrams. If you need complex routing, adjust the connection sides instead.

**Q: Controls are hidden**
A: They're now at the midpoint, but double-check the edge is selected

**Q: Segments are diagonal**
A: This shouldn't happen! The system enforces 90Â° angles. If you see diagonal segments, try resetting to auto route.
